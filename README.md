# EMA Kundenaufnahme App

Separate statische Web-App für Vercel.

## Hochladen auf Vercel
1. Ordner als neues GitHub Repository hochladen.
2. In Vercel: Add New Project → Repository auswählen.
3. Framework Preset: Other / Static.
4. Deploy klicken.

## iPhone
Safari öffnen → Vercel-Link öffnen → Teilen → Zum Home-Bildschirm.

Hinweis: GPS funktioniert nur über HTTPS. Vercel stellt HTTPS automatisch bereit.
